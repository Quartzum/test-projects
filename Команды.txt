python -m venv ... - создание вирутальной среды
pip install Django==4.0.4 - установка джанго

django-admin startproject ... - создание проекта в джанго

clear - очистить терминал

cd - переход из папки в папку

python manage.py runserver - запуск локального сервера

Ctrl+c - остановка сервера

python manage.py startapp ... - создание нового приложения

python manage.py makemigrations - создание миграции

python manage.py migrate  - выполнение миграции

python manage.py createsuperuser  - создание суперпользователя
