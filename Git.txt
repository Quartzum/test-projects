Git Bush:

git config --global user.name - посмотреть свое имя внутри Git.
git config --global user.name "Name Family" - изменить свое имя внутри Git.
git config --global user.email - посмотреть свой email внутри Git.
git config --global user.email "Some email" - изменить свой email внутри Git.



git --version - версия Git
git --help - команды Git
git init - инициализация пустого репозитория. C помощью этой команды создалась системная папка .git(ее не видно, но она есть)
cd .git - переход в скрытую папку гита.
cd .. - перейти на папку назад
git status - статус Гита(что с ним происходит в данный момент)
git add index.html - добавить в отслеживанье гитом файла(или группы файлов)
git add . - добавить все файлы в отслеживанье
git rm --cached index.html - удалить файл в отслеживанье
git commit -m "название коммита" - создание коммита
git branch - какие есть ветки
git branch sometext - создание ветки
git branch -D sometext - удаление ветки
git checkout sometext - переход на другую ветку
git checkout -b sometext - переход на другую ветку и одновременное ее создание
git merge somebranch - слияние ветки с веткой master. Здесь происходит не уничтожение другой ветки, а передача информации в мастер.

git remote add origin https://github.com/Quartzum/git-course.git - добавить источник на гитхабе репозитория.
git push -u origin master - отправить файлы внутрь репозитория на гитхаб(в данном случае на ветку master).

git push - отправить файлы внутрь репозитория(последнего).
git remote rm origin - удалить зарег.репозиторий


git clone https://github.com/Quartzum/git-course.git - клонирование репозитория(если тебе нужно скопировать репозиторий с гитхаба у другого человека или у себя).
git pull - скопировать все изменения с репозитория в папку своего проекта

То есть ветка не удаляется.
Если мы создаем какой-то файл на другой ветке, и переходим на другую, то на другой ветке этого файла уже не будет.

Все что, мы делаем с Git внутри терминала на своем ПК - это локальная работа.
Если мы работаем с GitHub, то мы перемещаем его в хранилище внутри интернета.
